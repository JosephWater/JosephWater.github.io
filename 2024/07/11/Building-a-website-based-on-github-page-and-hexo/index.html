基于github page和hexo建站的常见问题

搭建教程

这里参考了B站视频【Hexo+Github：个人博客的搭建】https://www.bilibili.com/video/BV16b4y1G7fo?vd_source=9d90f6ee40f2641963493d9b0f4e5110 和学姐的博客 https://blog.rainnn.top/article/c411194a-3699-4d6d-984c-b00c472179b4 已经讲的很详细了，这里不再赘述。

常见问题

首先建议使用powershell而不是视频中的git bash，不然可能会出现权限上的问题。

hexo的安装


解决方法：
在默认情况下，我们是无法执行powershell脚本的需要更改执行策略
win10下更改执行策略:
1.打开设置
2.搜索power
选择选项:允许本地powershell在不签名的情况下运行
3.勾选，点击应用就可以了

执行hexo d后报如下错


解决方法：这是因为没安装 hexo-deployer-git 插件，在站点目录下输入下面的插件安装就好了
npm install hexo-deployer-git--save

执行hexo d后仓库名旁没有出现小红（或者棕？）点，即没有自动部署

解决方法：

打开网站后css样式无效


解决方法：在配置文件_config.yml中修改url为githubID.github.io，去掉原文件中建议的/program
